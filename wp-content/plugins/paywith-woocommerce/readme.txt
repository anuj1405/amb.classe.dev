== Installation ==

= Installation Steps =
* Ensure you have latest version of WooCommerce plugin installed.
* Copy the folder named 'paywith-woocommerce' into the directory location /wp-content/plugins/. Or, you may choose to upload the paytm folder via the Woocommerce Admin panel.
* Activate the plugin through the left side 'Plugins' menu in WordPress.

= Configuration =
* Visit the WooCommerce => settings page, and click on the Checkout tab.
* Scroll down the Checkout page and go to the setting option of paytm under "Gateway Display".
* Click on paytm to edit the settings. If you do not see paytm in the list at the top of the screen make sure you have activated the plugin in * the WordPress Plugin Manager.
* Enable the Payment Method, name it Credit Card / Debit Card / Internet Banking (this will show up on the payment page your customer sees), add in your merchant id , secret key, paytm Gateway URL, Industry Type, ChannelID, Web Site, Return page, mode(0 for trial purpose and 1 for live purpose) and Click Save.
* Now you can see paytm in your payment option.
